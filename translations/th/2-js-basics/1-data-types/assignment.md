# การฝึกฝนประเภทข้อมูล

## คำแนะนำ

ลองจินตนาการว่าคุณกำลังสร้างตะกร้าสินค้า เขียนเอกสารเกี่ยวกับประเภทข้อมูลที่คุณจะต้องใช้เพื่อสร้างประสบการณ์การช้อปปิ้งของคุณ คุณเลือกประเภทข้อมูลเหล่านี้มาได้อย่างไร?
