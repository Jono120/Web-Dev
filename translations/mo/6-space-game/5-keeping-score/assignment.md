# 建立一個計分遊戲

## 指示

創建一個遊戲，以創意的方式顯示生命值和分數。一個建議是將生命值顯示為心形，並將分數顯示為螢幕底部中央的大數字。可以在這裡查看 [免費遊戲資源](https://www.kenney.nl/)

# 評分標準

| 標準     | 卓越表現               | 合格表現                   | 需要改進                  |
| -------- | ---------------------- | --------------------------- | -------------------------- |
|          | 完整的遊戲呈現         | 部分遊戲呈現               | 部分遊戲存在錯誤          |
