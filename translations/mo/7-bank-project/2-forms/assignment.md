# 為您的銀行應用程式設計樣式

## 指示

建立一個新的 `styles.css` 檔案，並在您目前的 `index.html` 檔案中加入連結。在您剛建立的 CSS 檔案中，添加一些樣式，使 *登入* 和 *儀表板* 頁面看起來整潔美觀。嘗試創建一個色彩主題，為您的應用程式打造自己的品牌形象。

> 提示：如果需要，您可以修改 HTML 並添加新的元素和類別。

## 評分標準

| 評分標準 | 卓越                                                                                                                   | 合格                                                                         | 需要改進                                                                                     |
| -------- | ----------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------- |
