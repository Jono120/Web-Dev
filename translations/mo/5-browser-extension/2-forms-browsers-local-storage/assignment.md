# 採用一個 API

## 說明

使用 API 可以非常有趣。這裡有一個[免費 API 的列表](https://github.com/public-apis/public-apis)。選擇一個 API，並建立一個可以解決問題的瀏覽器擴充功能。這個問題可以很小，例如缺少足夠的寵物圖片（那麼可以試試 [dog CEO API](https://dog.ceo/dog-api/)），也可以是更大的問題——盡情享受吧！
