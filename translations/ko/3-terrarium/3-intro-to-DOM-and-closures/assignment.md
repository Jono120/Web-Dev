# DOM과 조금 더 친해지기

## 지침

DOM에 대해 조금 더 조사하며 DOM 요소를 '채택'해 보세요. MDN의 [DOM 인터페이스 목록](https://developer.mozilla.org/docs/Web/API/Document_Object_Model)을 방문하여 하나를 선택하세요. 웹에서 해당 인터페이스가 사용된 사례를 찾아보고, 그것이 어떻게 사용되는지에 대한 설명을 작성하세요.
