# 게임 모형 만들기

## 지침

수업에서 제공된 코드 샘플을 사용하여, 당신이 즐기는 게임을 표현해 보세요. 간단한 게임이어야 하지만, 목표는 클래스 또는 구성 패턴과 pub/sub 패턴을 사용하여 게임이 어떻게 시작될 수 있는지를 보여주는 것입니다. 창의력을 발휘해 보세요!
