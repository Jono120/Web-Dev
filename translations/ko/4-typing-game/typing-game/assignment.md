# 새로운 키보드 게임 만들기

## 지침

키보드 이벤트를 사용하여 작업을 수행하는 작은 게임을 만들어보세요. 타이핑 게임일 수도 있고, 키 입력에 따라 화면에 픽셀을 그리는 예술적인 게임일 수도 있습니다. 창의력을 발휘해보세요!

| --------- | ------------------------ | ------------------------ | ----------------- |
|           | 완성된 게임이 제시됨     | 게임이 매우 간단함       | 게임에 버그가 있음 |
|           |                          |                          |                   |
