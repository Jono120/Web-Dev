# 은행 앱 스타일링하기

## 지침

새로운 `styles.css` 파일을 생성하고 현재 `index.html` 파일에 링크를 추가하세요. 방금 생성한 CSS 파일에 스타일을 추가하여 *로그인* 및 *대시보드* 페이지를 깔끔하고 정돈된 모습으로 만들어 보세요. 앱만의 브랜드를 나타낼 수 있는 색상 테마를 만들어 보세요.

> 팁: 필요하다면 HTML을 수정하고 새로운 요소와 클래스를 추가할 수 있습니다.
