# Refaktoriranje CSS-a

## Upute

Preoblikujte terarij koristeći ili Flexbox ili CSS Grid i napravite snimke zaslona kako biste pokazali da ste ga testirali na nekoliko preglednika. Možda ćete trebati promijeniti oznake, pa stvorite novu verziju aplikacije s umetnutom umjetnošću za vaš refaktor. Ne brinite o tome da elementi budu povlačivi; za sada samo refaktorirajte HTML i CSS.

## Rubrika

| Kriteriji | Izvrsno                                                          | Zadovoljavajuće               | Potrebno poboljšanje                 |

|           | Prikazati potpuno preoblikovan terarij koristeći Flexbox ili CSS Grid | Preoblikovati nekoliko elemenata | Ne uspjeti preoblikovati terarij uopće |
