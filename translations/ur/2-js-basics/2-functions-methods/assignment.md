# فنکشنز کے ساتھ مزہ

## ہدایات

مختلف فنکشنز بنائیں، ایسے فنکشنز بھی جو کچھ واپس کرتے ہیں اور ایسے بھی جو کچھ واپس نہیں کرتے۔

دیکھیں کہ کیا آپ ایسا فنکشن بنا سکتے ہیں جس میں عام پیرامیٹرز اور ڈیفالٹ ویلیوز والے پیرامیٹرز کا امتزاج ہو۔
