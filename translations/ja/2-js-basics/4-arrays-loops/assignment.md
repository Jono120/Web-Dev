# 配列をループする

## 手順

1から20までの数字のうち、3番目ごとの数字をリスト化し、それをコンソールに出力するプログラムを作成してください。

> TIP: forループを使用し、イテレーション式を変更してください

## 採点基準

| 基準     | 優秀                                   | 適切                     | 改善が必要                     |
| -------- | --------------------------------------- | ------------------------ | ------------------------------ |
|          | プログラムが正しく動作し、コメントがある | コメントがないプログラム | プログラムが不完全またはバグがある |
