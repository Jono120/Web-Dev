# 衝突を探る

## 手順

衝突の仕組みをよりよく理解するために、いくつかのアイテムが衝突する小さなゲームを作成してください。キー操作やマウスクリックでアイテムを動かし、衝突した際に何かが起こるようにしましょう。例えば、隕石が地球に衝突する場面や、バンパーカーのようなものを作ることができます。創造力を発揮してください！
