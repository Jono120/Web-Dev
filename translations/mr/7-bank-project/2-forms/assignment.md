# आपल्या बँक अ‍ॅपला स्टाइल करा

## सूचना

नवीन `styles.css` फाइल तयार करा आणि ती आपल्या सध्याच्या `index.html` फाइलमध्ये लिंक करा. तुम्ही तयार केलेल्या CSS फाइलमध्ये काही स्टाइलिंग जोडा जेणेकरून *Login* आणि *Dashboard* पृष्ठे आकर्षक आणि व्यवस्थित दिसतील. तुमच्या अ‍ॅपसाठी स्वतःचा ब्रँडिंग देण्यासाठी रंग थीम तयार करण्याचा प्रयत्न करा.

> टीप: तुम्ही HTML सुधारित करू शकता आणि आवश्यक असल्यास नवीन घटक आणि वर्ग जोडू शकता.
