# לאמץ API

## הוראות

APIs יכולים להיות מאוד מהנים לשימוש. הנה [רשימה של רבים מהם בחינם](https://github.com/public-apis/public-apis). בחרו API ובנו תוסף לדפדפן שמספק פתרון לבעיה. זה יכול להיות בעיה קטנה כמו חוסר בתמונות של חיות מחמד (לכן, נסו את [dog CEO API](https://dog.ceo/dog-api/)) או משהו גדול יותר - תהנו!
