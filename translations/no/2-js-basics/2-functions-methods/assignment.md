# Moro med funksjoner

## Instruksjoner

Lag forskjellige funksjoner, både funksjoner som returnerer noe og funksjoner som ikke returnerer noe.

Se om du kan lage en funksjon som har en blanding av parametere og parametere med standardverdier.
