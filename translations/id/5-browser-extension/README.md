# Membangun Ekstensi Browser

Membangun ekstensi browser adalah cara yang menyenangkan dan menarik untuk memikirkan kinerja aplikasi Anda sambil membuat jenis aset web yang berbeda. Modul ini mencakup pelajaran tentang cara kerja browser dan cara menerapkan ekstensi browser, cara membuat formulir, memanggil API, menggunakan penyimpanan lokal, serta cara mengukur kinerja situs web Anda dan meningkatkannya.

Anda akan membangun ekstensi browser yang dapat digunakan di Edge, Chrome, dan Firefox. Ekstensi ini, yang mirip dengan situs web mini yang dirancang untuk tugas yang sangat spesifik, memeriksa [C02 Signal API](https://www.co2signal.com) untuk penggunaan listrik dan intensitas karbon di suatu wilayah tertentu, lalu memberikan pembacaan jejak karbon wilayah tersebut.

Ekstensi ini dapat digunakan secara ad hoc oleh pengguna setelah kunci API dan kode wilayah dimasukkan ke dalam formulir untuk menentukan penggunaan listrik lokal, sehingga menawarkan data yang dapat memengaruhi keputusan listrik pengguna. Misalnya, mungkin lebih baik menunda penggunaan pengering pakaian (aktivitas yang intensif karbon) selama periode penggunaan listrik yang tinggi di wilayah Anda.
