# Dukungan

## Cara Melaporkan Masalah dan Mendapatkan Bantuan  

Proyek ini menggunakan GitHub Issues untuk melacak bug dan permintaan fitur. Harap cari masalah yang sudah ada sebelum membuat masalah baru untuk menghindari duplikasi. Untuk masalah baru, laporkan bug atau permintaan fitur Anda sebagai Issue baru.

Untuk bantuan dan pertanyaan tentang penggunaan proyek ini, silakan merujuk ke [panduan kontribusi kami](CONTRIBUTING.md).

## Kebijakan Dukungan Microsoft  

Dukungan untuk proyek ini terbatas pada sumber daya yang tercantum di atas.
