# Redefina o Estilo da sua Extensão

## Instruções

A base de código desta extensão já inclui estilos, mas não é obrigatório utilizá-los; personalize a sua extensão ao editar o ficheiro css e criar o seu próprio estilo.

| -------- | ------------------------------------------- | --------------------- | ------------------- |
