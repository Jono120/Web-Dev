# کار بیشتر با DOM

## دستورالعمل‌ها

کمی بیشتر درباره DOM تحقیق کنید و یک عنصر DOM را «انتخاب» کنید. به [فهرست رابط‌های DOM](https://developer.mozilla.org/docs/Web/API/Document_Object_Model) در MDN مراجعه کنید و یکی را انتخاب کنید. سپس ببینید که چگونه در یک وب‌سایت استفاده شده است و توضیحی درباره نحوه استفاده از آن بنویسید.
