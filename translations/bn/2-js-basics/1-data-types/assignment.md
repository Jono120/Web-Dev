# ডেটা টাইপ অনুশীলন

## নির্দেশাবলী

ধরুন আপনি একটি শপিং কার্ট তৈরি করছেন। সেই অভিজ্ঞতা সম্পূর্ণ করতে আপনার যেসব ডেটা টাইপ প্রয়োজন হবে, সেগুলোর উপর কিছু ডকুমেন্টেশন লিখুন। আপনি কীভাবে আপনার পছন্দগুলোতে পৌঁছালেন তা ব্যাখ্যা করুন।
