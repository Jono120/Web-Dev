# Muokkaa laajennuksesi tyyliä

## Ohjeet

Tämän laajennuksen koodipohja sisältää valmiita tyylejä, mutta sinun ei tarvitse käyttää niitä; tee laajennuksesta omannäköisesi muokkaamalla sen css-tiedostoa.

| -------- | ------------------------------------------- | --------------------- | ------------------ |
