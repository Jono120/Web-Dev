# إعادة تصميم الإضافة الخاصة بك

## التعليمات

يأتي الكود الأساسي لهذه الإضافة مع أنماط جاهزة، ولكن ليس عليك استخدامها؛ اجعل الإضافة خاصة بك عن طريق إعادة تصميمها وتحرير ملف css الخاص بها.
