# 改善路由

## 說明

目前的路由宣告僅包含模板 ID，但在顯示新頁面時，有時需要更多功能。讓我們透過以下兩個額外功能來改進路由實現：

- 為每個模板指定標題，並在模板變更時更新窗口標題為新標題。
- 增加一個選項，在模板變更後執行一些程式碼。我們希望每次顯示儀表板頁面時，在開發者主控台中打印 `'Dashboard is shown'`。
