# 練習你的 HTML：建立一個部落格模型

## 指導說明

想像你正在設計或重新設計你個人的網站。創建一個網站的圖形模型，然後寫下你將用來構建網站各個元素的 HTML 標記。你可以在紙上完成，然後掃描，或者使用你選擇的軟體，但請確保手動編寫 HTML 標記。
