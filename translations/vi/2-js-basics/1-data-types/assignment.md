# Thực Hành Kiểu Dữ Liệu

## Hướng Dẫn

Hãy tưởng tượng bạn đang xây dựng một giỏ hàng. Viết một số tài liệu về các kiểu dữ liệu mà bạn cần để hoàn thiện trải nghiệm mua sắm. Làm thế nào bạn đưa ra được các lựa chọn này?
