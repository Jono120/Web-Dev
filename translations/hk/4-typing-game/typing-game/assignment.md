# 建立一個新的鍵盤遊戲

## 指示

創建一個使用鍵盤事件來完成任務的小型遊戲。它可以是一種不同類型的打字遊戲，或者是一種藝術型遊戲，透過按鍵在螢幕上繪製像素。發揮創意！
