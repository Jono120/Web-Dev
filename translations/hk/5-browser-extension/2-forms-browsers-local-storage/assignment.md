# 採用 API

## 指引

使用 API 可以非常有趣。這裡有一個[免費 API 的列表](https://github.com/public-apis/public-apis)。選擇一個 API，並建立一個解決問題的瀏覽器擴展。這個問題可以很小，例如缺乏足夠的寵物圖片（可以試試 [dog CEO API](https://dog.ceo/dog-api/)），或者是更大的問題——盡情享受吧！
