# Исследуйте столкновения

## Инструкции

Чтобы лучше понять, как работают столкновения, создайте небольшую игру с несколькими объектами, которые сталкиваются. Сделайте так, чтобы они двигались с помощью нажатий клавиш или кликов мыши, и чтобы при столкновении с одним из объектов происходило какое-то действие. Это может быть, например, метеорит, ударяющий Землю, или машинки-аттракционы. Проявите креативность!

## Критерии оценки

| Критерий | Превосходно                                                                                                              | Удовлетворительно              | Требует улучшения |
| -------- | ------------------------------------------------------------------------------------------------------------------------ | ------------------------------ | ----------------- |
|          | Создан полностью рабочий пример кода, объекты нарисованы на холсте, происходит базовое столкновение, есть реакции        | Код неполный в каком-либо аспекте | Код работает некорректно |
